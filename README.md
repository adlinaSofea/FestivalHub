# FestivalHub

**Overview**

FestivalHub is a web-based festival event booking system developed to provide users with a centralized platform for discovering and booking festival events. The system allows users to browse available events, view event details, and manage bookings through a user-friendly interface.

The project was developed using Java Enterprise technologies and follows the Model-View-Controller (MVC) architectural pattern to ensure maintainability, scalability, and separation of concerns.

**Features**
1. User registration and authentication
2. Browse and search festival events
3. View detailed event information
4. Event booking management
5. Centralized event information platform

**Technologies Used**
**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Java Enterprise (Java EE)
- Database
- MySQL
- Java Persistence API (JPA)

**Software Engineering Practices**
- MVC Architecture
- Object-Oriented Programming (OOP)
- System Analysis and Design
